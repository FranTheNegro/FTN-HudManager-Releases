<h2 id="english">English</h2>

**Administration and Broadcasting Panel for Counter-Strike 2**

### Overview

**FTN HUD Manager** is a software tool designed for esports producers, tournament organizers, and broadcasting teams. It provides an advanced control panel to manage and customize Counter-Strike 2 overlays, as well as process live match data in real-time.

### Key Features

- **Game State Integration (GSI):** Real-time data capture directly from CS2. Ensures accurate reading of match events, player statistics, and round results with zero latency.
- **Centralized Administration Panel:** Unified interface to manage player information, rosters, scoreboards, team logos, and overall broadcast state.
- **Low-Latency Broadcasting:** Implementation of WebSockets (Socket.io) to send instantaneous updates to broadcasting software such as OBS Studio or vMix, without the need to reload sources.
- **Dynamic Customization:** Full control over HUD aesthetics, allowing live modifications of colors and styles to adapt to the corporate identity of any league or tournament.
- **Hybrid Data Management:** Architecture based on SQLite3 for high-speed local operations and Supabase support for cloud data management.
- **Optimized Architecture:** Developed with cutting-edge technologies (Electron, React, Vite, and TailwindCSS) to guarantee fluidity, stability, and efficient use of system resources.

<h2 id="español">Español</h2>

**Panel de Administración y Broadcasting para Counter-Strike 2**

### Descripción General

**FTN HUD Manager** es una herramienta de software diseñada para productores de esports, organizadores de torneos y equipos de transmisión. Proporciona un panel de control avanzado para gestionar y personalizar overlays de Counter-Strike 2, además de procesar los datos de las partidas en tiempo real.

### Características Principales

- **Integración GSI (Game State Integration):** Captura de datos en tiempo real directamente desde CS2. Asegura una lectura precisa de los eventos de la partida, estadísticas de jugadores y resultados de ronda sin latencia.
- **Panel de Administración Centralizado:** Interfaz unificada para gestionar información de jugadores, alineaciones, marcadores, logos de equipos y el estado general de la transmisión.
- **Transmisión de Baja Latencia:** Implementación de WebSockets (Socket.io) para enviar actualizaciones instantáneas hacia software de transmisión como OBS Studio o vMix, sin necesidad de recargar fuentes.
- **Personalización Dinámica:** Control total sobre la estética del HUD, permitiendo modificaciones en vivo de colores y estilos para adaptarse a la identidad corporativa de cualquier liga o torneo.
- **Gestión Híbrida de Datos:** Arquitectura basada en SQLite3 para operaciones locales de alta velocidad y soporte de Supabase para la gestión de datos en la nube.
- **Arquitectura Optimizada:** Desarrollado con tecnologías de vanguardia (Electron, React, Vite y TailwindCSS) para garantizar fluidez, estabilidad y un uso eficiente de los recursos del sistema.
